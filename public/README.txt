РАСПИСАНИЕ СМЕН - ИНСТРУКЦИЯ ПО ЗАПУСКУ
========================================

ВАЖНО! Просто открыть index.html в браузере НЕ ПОЛУЧИТСЯ.
Современные веб-приложения требуют запуска через веб-сервер.

═══════════════════════════════════════════════════════════

СПОСОБ 1: Python (САМЫЙ ПРОСТОЙ)
---------------------------------
1. Откройте командную строку (Windows: Win+R → cmd)
   или терминал (Mac/Linux)

2. Перейдите в папку с сайтом:
   cd путь\к\этой\папке

3. Запустите команду:
   python -m http.server 8000

4. Откройте в браузере:
   http://localhost:8000

═══════════════════════════════════════════════════════════

СПОСОБ 2: Node.js
-----------------
1. Установите Node.js с сайта https://nodejs.org

2. Откройте командную строку в папке с сайтом

3. Запустите команду:
   npx serve .

4. Откройте адрес из терминала (обычно http://localhost:3000)

═══════════════════════════════════════════════════════════

СПОСОБ 3: Установка как приложение (PWA)
-----------------------------------------
1. Запустите сайт способом 1 или 2

2. Откройте в браузере Chrome или Edge

3. Нажмите значок установки в адресной строке (⊕)

4. После установки приложение будет работать БЕЗ ИНТЕРНЕТА!

═══════════════════════════════════════════════════════════
